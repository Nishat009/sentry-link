Here are the **run steps in a simple list format**:

1. Clone the repository
   git clone https://github.com/Nishat009/sentry-link.git
2. Go to the project folder
   cd sentry-link
3. Install dependencies
   npm install
4. Start the development server
   npm run dev
5. Open in browser
   http://localhost:8080
